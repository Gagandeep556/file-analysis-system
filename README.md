# File Analysis system

## Description

It is the text file analysis system.Backen syatem is made using express js as framework and database used is mysql with sequelize(ORM).

## Installation

```bash
npm install
```

## Running

```bash
npm start
```

## Env

```bash
PORT=8080
DB_NAME=Task
DB_USER=
DB_PASS=
DB_HOST=
```
